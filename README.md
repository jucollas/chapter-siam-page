# 📐 SIAM Cali Student Chapter — Landing Page

Official landing page for the **SIAM Cali Student Chapter**, designed to present the community, its research areas, members, activities, and contact channels.

---

## 🌐 Overview

This project is a **static landing page** aimed at:

* Showcasing the mission and vision of the chapter
* Presenting research and interest areas
* Highlighting members and faculty
* Promoting academic and community activities
* Providing a contact point for prospective members

The design follows an **academic and editorial style**, emphasizing typography, visual hierarchy, and clarity.

---

## 🛠️ Tech Stack

* ⚡ **Astro** — main framework
* 🎨 **Tailwind CSS (v4)** — styling and design system
* ☁️ **Vercel** — deployment

---

## 📁 Project Structure

```bash
src/
 ├── components/     # Reusable UI components
 ├── layouts/        # Main layout(s)
 ├── pages/          # Application routes
 └── styles/         # Global styles (Tailwind)

public/              # Static assets
```

## 🎨 Design System

The project uses a **design token approach (Tailwind v4)** defined in CSS:

* Custom color palette (`navy`, `accent`, `cream`, etc.)
* Serif and sans-serif typography system
* Optimized reading width (`~70ch`)

Example:

```html
<div class="bg-navy text-cream">
```
